Памятка для участников проекта

Каждый сайт использует свой набор терминов, выражений, понятий, которые как бы понятные для каждого, но в действительности в сознании проявляется свое восприятие.

Мы находимся под воздействием постоянно меняющихся условий, в результате наше восприятие меняется.

Многие термины, понятия, взгляды, которые раньше были как бы нейтральными, теперь становятся значимыми, в худшем отталкивающими, настораживающими, а зачастую вызывающими тревогу, опасение.

По этой причине терминология, наименования текстовых ссылок отличаются от общепринятых.
Например, вместо личный кабинет - используется профиль.
вместо регистрация пользователя - используется создать профиль.

В основе принципа подачи информации минимализм. Т.е. как можно меньше информации. И только, если появляется интерес, пользователь может сделать переход по текстовой ссылке,
а там, где это целесообразно, вместо текстовых ссылок используются кнопки.

Структура сайта

Баннер - верхняя часть - предназначен для быстрого восприятия информации всего сайта,
Навигационная панель - выделена зеленым цветом содержит текстовые ссылки для перехода по сайту
Средняя часть информационная, предназначена для демонстрации начальной, значимой информации
Нижняя часть сайта содержит данные для контактов

Назначение текстовых ссылок навигационной панели
  • Главная текстовая ссылка переход на ГЛАВНУЮ страницу сайта.
  • Контакты переход на страницу создания сообщения на сайт (обратная связь)
  • Профиль переход на страницу, где будут отображены данные, которые используются
    для отображения значимой для вас информации
  • О нас переход на страницу сведений о проекте, основная идея при создании.
  • Вопросы переход на страницу, которая позволит развеять сомнения относительно продукции, также получить ответы по наиболее часто возникающим вопросам из СМИ, Internet и т.д.
  • Админ-пан переход на страницу для руководителей групп
  • Выход переход на страницу, повторной авторизации, под другим логином или иной причине
